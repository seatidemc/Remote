Remote

Remote 是一个可用于从 Web 与 Minecraft 服务器交互的 Bukkit 插件。这里是它的默认页面。

Implemented API

get /device/constant/:type

get /device/dynamic/:type

get /all-info

post /server/exec

ws /server/console

Complete information on how to call them is shown in Wiki (Chinese).
